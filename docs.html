Install CLI

Requirements

  • Windows 10, Mac OSX, Linux
  • Node.js v12 or higher

Install

Mac (Homebrew)

$ brew install weborama-cli

NPM

$ npm install weborama-cli

Yarn

$ yarn install weborama-cli
